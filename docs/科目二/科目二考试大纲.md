# 科目二：教育知识与能力考试大纲

## 一、考试核心内容

1. 教育基础理论
   - 教育的定义、目的、功能
   - 教育与人的全面发展
   - 中学教育特点
   - 素质教育理念与教育现代化

2. 教育法律法规与教师职业道德
   - 教师法、教育法、义务教育法、未成年人保护法
   - 教师职业道德规范及行为要求
   - 教师职责、权利与法律责任
   - 法律法规适用场景与案例判断

3. 中学课程与教学
   - 课程性质、课程目标与课程结构
   - 教学原则、教学方法、教学手段
   - 教学过程与组织形式
   - 教学设计四要素：目标、内容、方法、评价
   - 教学评价类型、功能与应用

4. 中学生学习心理与发展心理
   - 学习动机、学习兴趣、学习态度
   - 认知发展特征与学习方式
   - 情绪、意志、自我管理与个体差异
   - 心理辅导策略与学习支持

5. 中学德育与班级管理
   - 德育目标、德育方法与德育活动
   - 班级规范建设与班集体氛围营造
   - 班级管理原则与常见策略
   - 家校合作与师生关系处理

6. 教师职业素养与教师心理
   - 教师职业理念与职业使命
   - 教师角色定位与专业发展
   - 教师心理健康、压力调适与自我反思

## 二、题型与考法

### 1. 客观题
- 单项选择题
- 多项选择题
- 判断题
- 名词解释/术语解释

### 2. 主观题
- 简答题：解释概念、说明原理、提出建议
- 材料分析题：分析教育情境、给出对策
- 案例分析题：结合理论分析问题、提出解决方案
- 教学设计题：按目标、重难点、过程、评价结构设计方案
- 综合题：教育基础、心理、德育等内容融合考查

## 三、重难点与复习建议

### 1. 重难点内容
- 教学设计四要素与教学评价体系
- 教学原则与教学方法匹配
- 学习动机与认知发展特征
- 班级管理策略与德育实施方法
- 法规适用场景与教师职业道德判断

### 2. 高频考法
- 选择题：基础概念与理论知识
- 材料/案例题：理论应用与方案设计
- 教学设计题：教学逻辑与结构完整性
- 法规与职业道德题：情境判断与处理建议

### 3. 复习顺序建议
1. 先攻“中学课程与教学”与“教育基础与法规”
2. 再补“中学生学习心理与发展心理”
3. 然后学“中学德育与班级管理”
4. 最后巩固“教师职业素养与教师心理”

### 4. 复习方法
- 构建知识树：基础理论 → 教学设计 → 心理特征 → 管理策略
- 重点练习材料分析和案例题
- 教学设计题反复练“目标、内容、方法、评价”
- 法规题记忆法律名称和核心条款精神
