# 一、材料分析最常考四大模块
1. **教学规律、教学原则、教学方法**（考得最多）
2. **德育规律、德育原则、德育方法**
3. **学习动机、学习迁移、遗忘、归因**
4. **班级管理、班主任工作、师生关系**

---

# 二、教学类（必考）
## 1. 教学过程基本规律（看到“教与学”直接套）
1. **间接经验与直接经验相统一**
2. **掌握知识与发展能力相统一**
3. **教师主导与学生主体相统一**
4. **传授知识与思想品德教育相统一（教育性规律）**

## 2. 八大教学原则（材料题万能金句）
1. **启发性原则**
   关键词：引导思考、提问、鼓励探究、不直接给答案
2. **因材施教原则**
   关键词：差异、不同学生不同方法、个别对待
3. **循序渐进原则**
   关键词：由浅入深、由易到难、不跳跃
4. **理论联系实际原则**
   关键词：联系生活、解决实际问题
5. **直观性原则**
   关键词：实物、模型、演示、多媒体
6. **巩固性原则**
   关键词：复习、练习、及时巩固
7. **科学性与思想性统一**
   关键词：教书+育人、品德渗透
8. **量力性原则**
   关键词：难度适中、符合学生水平

## 3. 新课改教学观（材料出现“老师怎么教”必写）
- 教学从**教育者为中心**转向**学习者为中心**
- 从**教会学生知识**转向**教会学生学习**
- 从**重结论轻过程**转向**重结论更重过程**
- 从**关注学科**转向**关注人**

## 4. 教师角色转变
- 学习的**引导者、促进者**
- 课程的**建设者、开发者**
- 教育教学的**研究者**
- 社区型开放教师

# 三、德育类（高频第二）
## 1. 德育过程规律
1. 知情意行**全面发展**，具有**多端性**
2. 在**活动与交往**中形成品德
3. 长期、反复、逐步提高
4. 促进学生**思想内部矛盾转化**

## 2. 德育原则（材料出现“学生犯错、早恋、叛逆”直接用）
1. **疏导原则**：循循善诱、以理服人
2. **尊重信任与严格要求相结合**
3. **因材施教**：根据个性特点教育
4. **长善救失**：发扬优点，克服缺点
5. **知行统一**：言行一致
6. **教育一致性与连贯性**：家校社配合
7. **正面教育与纪律约束相结合**

## 3. 德育方法
- 说服教育
- 榜样示范
- 情感陶冶
- 实际锻炼
- 品德评价（奖惩）
- 自我修养

---

# 四、学习心理类（常出现在“学生学不好”材料）
## 1. 学习动机
- 动机**适中**效率最高（耶克斯-多德森）
- 培养：目标、兴趣、归因、及时反馈
- 激发：问题情境、奖惩、合作竞争

## 2. 归因理论（学生成绩差、自卑）
引导归因于**努力**（内部、可控）
避免归因于**能力**（会导致习得性无助）

## 3. 遗忘与复习
- 先快后慢，及时复习、分散复习、过度学习

## 4. 迁移
- 正迁移/负迁移
- 促进迁移：总结规律、举一反三、联系旧知

## 5. 建构主义学习观
- 主动建构、社会互动、情境性学习

---

# 五、班级管理与班主任（几乎每套都考）
## 1. 班集体培养方法
1. 确定共同目标
2. 建立班干部核心
3. 建立正常秩序
4. 组织班会与活动
5. 培养正确舆论和良好班风

## 2. 班主任工作内容
- 了解研究学生
- 组织培养班集体
- 建立学生档案
- 操行评定
- 协调家校社教育
- 个别教育（后进生、优等生）

## 3. 后进生教育（必考）
- 关心热爱与严格要求
- 培养激发学习动机
- 因材施教
- 反复抓、抓反复
- 扬长避短

## 4. 师生关系
- 民主平等
- 尊师爱生
- 心理相容
- 教学相长

# 六、材料分析**万能答题模板**（必背）
1. **总：**该老师做法正确/错误，符合/违背了……
2. **分点：**
   ① 原理名称 + 解释
   ② 结合材料：材料中老师……体现了……
3. **总：**因此，教师应……

只要**原理写对+结合材料一句**，就能拿高分。
