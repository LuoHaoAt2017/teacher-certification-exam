## 科目二

### 一、简答题·必背24题（精简版，背就得分）
1. **教育的社会属性**
永恒性、历史性、相对独立性、阶级性（阶级社会）、生产性、民族性。

2. **影响人身心发展的因素**
遗传（前提）、环境（外部条件）、学校教育（主导）、主观能动性（决定）。

3. **我国教育目的基本精神**
培养社会主义建设者和接班人；德智体美劳全面发展；强调创新与实践；适应终身发展。

4. **课程内容三种文本形式**
课程计划、课程标准、教材。

5. **新课改三大学习方式**
自主学习、合作学习、探究学习。

6. **教学过程四大基本规律**
间接经验与直接经验统一；掌握知识与发展能力统一；教师主导与学生主体统一；传授知识与思想品德统一。

7. **八大教学原则（记关键词）**
科学性与思想性统一、直观性、启发性、循序渐进、巩固性、量力性、因材施教、理论联系实际。

8. **教学工作基本环节**
备课、上课、作业布置与批改、课外辅导、学业成绩检查与评定。

9. **一堂好课的基本标准**
目标明确、重点突出、内容正确、方法得当、表达清晰、组织严密、气氛热烈、板书有序。

10. **德育过程四大规律**
知情意行统一；长期反复逐步提高；活动交往为基础；内部矛盾转化。

11. **德育六大原则**
疏导原则、尊重与严格要求、因材施教、知行统一、正面教育与纪律约束、教育一致性与连贯性。

12. **德育常用方法**
说服教育、榜样示范、实际锻炼、情感陶冶、品德评价、自我修养。

13. **马斯洛需要层次**
生理→安全→归属与爱→尊重→求知→审美→自我实现。

14. **影响遗忘的因素**
时间、材料性质、数量、学习程度、位置效应、加工深度、情绪动机。

15. **有效防止遗忘（如何复习）**
及时复习、合理分配时间、反复阅读与尝试回忆、多样化复习、过度学习。

16. **学习动机与效果关系**
倒U型；中等强度动机效果最好；任务难宜低动机，任务简易高动机。

17. **促进学习迁移的策略**
精选教材、合理编排内容、科学编排程序、教授学习策略、提高迁移意识。

18. **皮亚杰认知发展四阶段**
感知运动（0-2）、前运算（2-7）、具体运算（7-11）、形式运算（11+）。

19. **维果茨基：最近发展区**
现有水平与可能发展水平之间的差距；教学应走在发展前面。

20. **气质类型及教育**
胆汁质、多血质、黏液质、抑郁质。因材施教，扬长避短，发扬积极品质。

21. **后进生教育策略**
尊重关爱、激发兴趣、因材施教、持之以恒、家校合作、发挥闪光点。

22. **班主任工作内容**
了解学生、组织培养班集体、建立学生档案、个别教育、班会活动、协调各方、操行评定、计划总结。

23. **培养班集体方法**
确定目标、建立核心队伍、建立秩序、组织活动、培养正确舆论与班风。

24. **教师成长三阶段**
关注生存、关注情境、关注学生。

### 三、辨析题·高频必考点（必考几类）
1. **负强化 ≠ 惩罚**
负强化：撤销厌恶刺激 → 行为增加
惩罚：呈现厌恶刺激 → 行为减少

2. **动机越强，效果越好（错）**
动机与效果呈**倒U型**，中等强度最佳。

3. **遗传/环境/教育决定发展（错）**
遗传是前提，环境是条件，教育主导，主观能动性决定。

4. **非正式群体都是消极的（错）**
分积极、消极、破坏型，要引导利用。

5. **德育 = 思想品德课（错）**
德育贯穿各科教学、活动、管理全过程，不只是一门课。

6. **注意转移 = 注意分散（错）**
转移：主动、有目的；分散：被动、分心。

7. **技能 = 知识（错）**
知识是认知经验，技能是活动方式，需反复练习形成。

### 四、材料分析题·万能套话（直接抄）
#### 教育学套话
- 体现**教师主导、学生主体**
- 遵循**启发性/因材施教/循序渐进**教学原则
- 采用**讨论法/演示法/探究法**，调动积极性
- 落实**立德树人**，促进全面发展
- 合理组织课堂，关注个体差异

#### 心理学套话
- 激发学习动机，培养兴趣
- 利用**最近发展区**，支架式教学
- 合理运用**正强化、及时评价**
- 关注学生情绪情感，促进心理健康
- 利用迁移规律，举一反三

#### 答题结构
1. 总体评价：该教师做法合理/不当
2. 分点对应理论：体现了XX原则/XX理论
3. 结合材料：文中……体现了……
4. 总结/启示：今后应……

## 科目三

### 一、先讲透：科目三对你来说，难度极低
科目三结构：
- **学科专业知识 43%** → 你几乎**白送分**
- **教学能力 57%** → 这是你唯一要学的部分

所以你的策略：
**专业知识不花时间，只猛攻教学能力。**

---

### 二、科目三·30天稳过计划（每天1～1.5小时即可）
#### 阶段1：前7天——抓框架+新课标（最关键）
目标：搞懂“信息技术课到底怎么教”
只学三件事：
1. **新课标四大核心素养（必背）**
   信息意识、计算思维、数字化学习与创新、信息社会责任
2. **高中信息技术两本必修结构**
   - 必修1：数据与计算（Python、算法、数据）
   - 必修2：信息系统与社会（网络、系统、安全、社会）
3. **教学方法背3个**
   任务驱动法、项目式学习、探究学习

> 这7天不用写代码、不用复习计算机基础，你都会。

---

#### 阶段2：第8～18天——猛攻教学设计（占分最高）
目标：**会写一篇完整教案，考场直接默写**
每天任务：
1. 背熟我给你的**教学设计万能模板**
2. 每2天写1篇完整教案，共写5篇即可，主题固定：
   - Python 基本语法
   - 条件语句 if
   - 循环 for/while
   - 数据处理与列表
   - 信息安全与网络道德
3. 学会写：
   - 核心素养目标
   - 重难点
   - 教学过程（导入→任务→实操→评价→小结）

你有开发经验，写实操环节**天然比师范生强**。

---

#### 阶段3：第19～25天——案例分析+刷真题
目标：选择题稳、案例题会套话
1. **刷近5年真题选择题**
   你大概正确率70%起步，遇到不会的点查一下即可
   重点看：
   - 进制、IP、网络
   - 数据库基本概念
   - Python基础语法题
   - 多媒体、信息安全
2. **案例分析背模板**
   看到案例就从这几个角度评价：
   - 是否体现核心素养
   - 是否任务驱动、学生动手
   - 是否分层教学、关注差异
   - 是否有评价、有小结
   - 教学环节是否完整

---

#### 阶段4：第26～30天——全真冲刺+默写
1. 整套真题限时做1～2套
2. **教学设计模板默写2遍**
3. 核心素养+教学术语背熟
4. 错题快速过一遍

完全足够稳过。
