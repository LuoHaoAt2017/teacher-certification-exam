# 科目一：综合素质考试大纲

## 一、考试核心内容

1. 职业理念与教师形象
   - 教师职业的性质与使命
   - 教师职业道德规范
   - 教师形象与职业行为
   - 教师与学生、家长、同事的关系

2. 教育法律法规与政策
   - 教师法、教育法、义务教育法
   - 未成年人保护法、预防未成年人犯罪法
   - 教育公平、教育机会均等、义务教育均衡发展
   - 教师权利与义务、学校教育责任

3. 教育文化素养
   - 中国传统文化与优秀文化教育价值
   - 现代文明与科学精神
   - 中华优秀传统文化的教育意义
   - 社会主义核心价值观与中学教育

4. 基本能力
   - 语言表达能力：阅读理解、文字表达、材料分析
   - 信息处理能力：资料筛选、归纳总结、判断推理
   - 逻辑思维能力：问题识别、概括归纳、评价判断
   - 解决问题能力：分析问题、制定方案、提出建议

5. 教师专业发展与自我提升
   - 教师继续教育与专业成长路径
   - 职业压力管理与心理调适
   - 理论联系实践、反思性实践意识

## 二、题型与考法

### 1. 客观题
- 单项选择题：基础概念、法规条款、职业理念判断
- 多项选择题：多角度分析、案例特征辨析
- 判断题：教育理念、职业行为、法律责任判断

### 2. 主观题
- 简答题：概念解释、原则说明、策略建议
- 材料分析题：给出教育情境材料，分析问题、提出对策
- 论述题：结合教师职业素养与教育理念开展阐述
- 案例题：教师职业道德、教育法律适用、班级常见问题处理

## 三、考法特点

1. 强调教师职业道德与法律法规结合
   - 多采用情境化案例进行考查
   - 注重法律法规实际应用与行为判断

2. 注重文化素养与价值观表达
   - 题目中常结合社会主义核心价值观、优秀传统文化
   - 考察教师价值理念、教育目标和学生成长关注点

3. 基本能力题型广泛
   - 材料分析题和阅读理解题占比高
   - 通过案例与资料考察归纳总结、文字表达、逻辑判断能力

4. 备考建议
   - 先熟悉题型和高频知识点
   - 注重法律法规条款的记忆与理解
   - 结合历年材料题练“读题-分析-写作”过程
   - 强化教师职业道德行为案例判断训练
