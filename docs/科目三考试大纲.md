# 科目三：信息技术学科知识与教学能力考试大纲

## 一、考试核心内容

1. 信息技术基础知识
   - 计算机系统组成与性能指标
   - 操作系统与常用软件功能
   - 网络基础与互联网应用
   - 数据处理与数据库基础
   - 信息安全与信息伦理

2. 信息技术学科课程标准与教学理念
   - 信息技术课程目标与课程定位
   - 信息技术核心素养与学科价值
   - 教学目标体系（知识与技能、过程与方法、情感态度与价值观）
   - 信息技术课程教学特点与课程改革趋势

3. 教学设计与教学实施
   - 教学设计四要素：目标、内容、方法、评价
   - 教学目标拆解、教学重点与教学难点分析
   - 教学方法与教学手段：直观性、参与性、探究性
   - 教学过程组织：导入、发展、巩固、课堂反馈
   - 教学评价方式：形成性评价、终结性评价、过程性评价

4. 信息技术课堂教学资源与教学手段
   - 多媒体教学设计与课堂呈现
   - 信息技术实验操作与实践活动设计
   - 学生信息素养培养与探究性学习
   - 教学资源开发与有效利用

5. 信息技术教学案例与课堂问题解决
   - 学生学习困难分析与差异化指导
   - 教学案例分析与改进建议
   - 课堂管理与信息技术课堂安全
   - 学科教学与综合实践活动融合

6. 学科前沿与教育应用
   - 人工智能、云计算、大数据、物联网等技术概述
   - 信息技术在教育教学中的应用场景
   - 信息技术与学科整合、创新教学模式

## 二、考试大纲要点

1. 重点考查信息技术学科的基础概念与应用能力
   - 计算机系统、网络、数据处理、软件应用
   - 信息技术与社会发展、信息伦理、信息安全

2. 着重评估教学设计能力与课堂教学组织能力
   - 教学设计方案的结构完整性与逻辑严密性
   - 教学目标明确、重难点突出、教学方法合理
   - 教学评价可操作、学习任务安排科学

3. 强调教学案例分析与课堂问题解决能力
   - 课堂教学情境分析与改进策略
   - 学生学习问题诊断与差异化教学建议
   - 教学资源与教学手段的有效结合

4. 关注学科前沿与信息技术教育现实应用
   - 结合高中信息技术课程标准与教材要求
   - 对技术应用于教学、学生素养培养有一定理解

## 三、题型与考法

### 1. 客观题
- 单项选择题：基础概念、术语、原理判断
- 多项选择题：多因素综合分析、技术特征辨析
- 判断题：基本原理与应用场景的准确性

### 2. 主观题
- 简答题：概念解释、技术特点、教材分析
- 教学设计题：按目标、内容、方法、评价设计教学方案
- 案例分析题：课堂情境问题分析、改进措施建议
- 材料题：结合材料说明教学理念、教学策略、评价方式
- 综合题：学科知识、教学策略与课程标准融合考查

### 3. 高频考法
- 选择题常考技术基础、概念区别、功能应用
- 简答题常考课程目标、教学策略、学科特点
- 教学设计题是核心题型，考查教学结构与实践能力
- 案例题常考课堂问题、学生学习支持、教学改进方案
- 关注教材章节与课程标准中的核心概念

## 四、重难点与复习建议

### 1. 重难点内容
- 计算机系统与网络基础概念
- 数据处理与信息安全基础
- 信息技术课程目标与核心素养
- 教学设计四要素与课堂组织策略
- 教学评价类型与方法
- 信息技术课堂案例分析与学生问题解决

### 2. 复习建议
- 先夯实“技术基础 + 课程标准”知识框架
- 用“教材章节 + 课程标准”构建教学设计模板
- 重点练习教学设计题和案例分析题
- 多看实际教学情境，联系教学目标与评价方案
- 深化对信息技术前沿应用在高中课堂中的理解
