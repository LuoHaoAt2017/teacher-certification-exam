# 科目三·信息技术学科知识高频考点速记清单

## 一、计算机网络（高频选择题）

### 1. OSI七层模型 vs TCP/IP四层模型

| OSI七层 | TCP/IP四层 | 典型协议 |
|---------|-----------|---------|
| 应用层 | 应用层 | HTTP、FTP、SMTP、DNS |
| 表示层 | ↑ | — |
| 会话层 | ↑ | — |
| 传输层 | 传输层 | **TCP**（可靠）、**UDP**（不可靠） |
| 网络层 | 网络层 | **IP**、ICMP、ARP |
| 数据链路层 | 网络接口层 | 以太网、PPP |
| 物理层 | ↑ | — |

**速记口诀**：物链网传会表应 → TCP/IP合为四层

### 2. IP地址分类

| 类别 | 范围 | 默认子网掩码 | 用途 |
|------|------|------------|------|
| A类 | 1.0.0.0~126.255.255.255 | 255.0.0.0 | 大型网络 |
| B类 | 128.0.0.0~191.255.255.255 | 255.255.0.0 | 中型网络 |
| C类 | 192.0.0.0~223.255.255.255 | 255.255.255.0 | 小型网络 |

**特殊IP**：
- 127.0.0.1 → 回送地址（localhost）
- 192.168.x.x、10.x.x.x、172.16~31.x.x → 私有地址
- 255.255.255.255 → 广播地址

### 3. 常见协议端口

| 协议 | 端口 | 功能 |
|------|------|------|
| HTTP | 80 | 网页传输 |
| HTTPS | 443 | 加密网页传输 |
| FTP | 21 | 文件传输 |
| SMTP | 25 | 发送邮件 |
| POP3 | 110 | 接收邮件 |
| DNS | 53 | 域名解析 |

### 4. 网络拓扑结构
- **总线型**：所有设备连一条线，成本低，单点故障影响大
- **星型**：中心节点连接所有设备，**最常用**
- **环型**：首尾相连，单向传输
- **网状型**：任意两节点相连，可靠性最高，成本最高

## 二、算法与程序设计（高频选择题+简答）

### 1. 三种基本结构
- **顺序结构**：从上到下依次执行
- **选择结构**：if/else（双分支）、switch（多分支）
- **循环结构**：for（计数型）、while（条件型）

### 2. 常见排序算法

| 算法 | 思想 | 平均时间复杂度 | 稳定性 |
|------|------|--------------|--------|
| 冒泡排序 | 相邻交换 | O(n²) | 稳定 |
| 选择排序 | 每轮选最小 | O(n²) | 不稳定 |
| 插入排序 | 插入有序序列 | O(n²) | 稳定 |
| 快速排序 | 分治+基准 | **O(n log n)** | 不稳定 |

### 3. 二叉树遍历

三种遍历顺序（**常考选择题**）：

```
      A
     / \
    B   C
   / \   \
  D   E   F

前序（根左右）: A B D E C F
中序（左根右）: D B E A C F
后序（左右根）: D E B F C A
```

**速记口诀**：前根中根后根，左右位置不变（前=根在前，中=根在中，后=根在后）

### 4. 栈和队列
- **栈**：先进后出（FILO），类比"弹匣"
- **队列**：先进先出（FIFO），类比"排队"

## 三、数据库技术（高频选择）

### 1. 关系模型三要素
- **关系**：一张二维表
- **元组**：一行记录
- **属性**：一列字段

### 2. 关系代数（必考）
| 运算 | 符号 | 含义 |
|------|------|------|
| 选择 | σ | 选**行**（满足条件） |
| 投影 | π | 选**列**（指定属性） |
| 连接 | ⨝ | 合并两个表 |
| 并 | ∪ | 合并行（去重） |

**易错点**：σ是选行，π是选列，不要混淆

### 3. SQL基本语句
```sql
SELECT 列名 FROM 表名 WHERE 条件 GROUP BY 分组 HAVING 筛选 ORDER BY 排序
```
**执行顺序**：FROM → WHERE → GROUP BY → HAVING → SELECT → ORDER BY

### 4. 主键 vs 外键
- **主键**：唯一标识一条记录，不能为空
- **外键**：引用另一张表的主键，建立表间联系

## 五、多媒体技术（低频选择）

| 类型 | 常见格式 |
|------|---------|
| 图像 | BMP（无压缩）、JPEG（有损）、PNG（无损/透明）、GIF（动态） |
| 音频 | WAV（无损）、MP3（有损）、MIDI（指令序列） |
| 视频 | AVI、MP4、MOV、FLV |

**位图 vs 矢量图**
- **位图**：像素构成，放大失真，如BMP、JPEG
- **矢量图**：数学公式构成，放大不失真，如SVG、CDR

**存储容量计算**：
- 图像：分辨率 × 颜色深度 ÷ 8（字节）
- 音频：采样频率 × 量化位数 × 声道数 × 时长（秒）

## 六、数据处理与信息安全（中频选择+简答）

### 1. 数据编码
- **ASCII码**：7位编码，128个字符
- **GB2312/GBK**：中文编码
- **Unicode/UTF-8**：国际统一编码

### 2. 进制转换
```
二进制 → 十进制：按权展开求和
十进制 → 二进制：除2取余倒序排列
```

### 3. 信息安全
- **病毒特征**：传染性、隐蔽性、破坏性
- **加密方式**：对称加密（DES/AES，密钥相同）、非对称加密（RSA，公钥+私钥）
- **防火墙**：阻止外部非法访问
- **数字签名**：确认身份+防篡改（用私钥签名，公钥验证）

## 七、计算机系统基础（低频选择）

### 1. 硬件组成
```
CPU（运算器+控制器）+ 存储器（内存+外存）+ 输入/输出设备
```

### 2. 存储层次（速度从快到慢，容量从小到大）
```
寄存器 > Cache（高速缓存）> RAM（内存）> 硬盘
```

### 3. 软件分类
- **系统软件**：操作系统、编译程序、数据库管理系统
- **应用软件**：Office、浏览器、Photoshop

| 概念 | 核心要点 |
|------|---------|
| **物联网（IoT）** | 物物相连，感知层+网络层+应用层 |

## 九、高频简答题考点总结

1. **信息与数据的区别**：数据是原始记录，信息是加工后有意义的数据
2. **信息的特征**：传递性、共享性、依附性、时效性、可处理性
3. **算法的特征**：有穷性、确定性、可行性、有0个或多个输入、有1个或多个输出
4. **TCP与UDP区别**：TCP面向连接（可靠）、UDP无连接（快速）
5. **结构化程序设计三种结构**：顺序、选择、循环
6. **人工智能在生活中的应用**：人脸识别、语音助手、推荐系统、自动驾驶
