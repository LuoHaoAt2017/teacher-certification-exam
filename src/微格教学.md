# 微格教学（Microteaching）

## 核心定义

微格教学，又称**微型教学**或**录像反馈教学**，是师范生和在职教师训练教学技能的一种方法。在**小范围**（5-10人）、**短时间**（5-10分钟）、**小内容**（一个单项技能）的条件下，模拟真实课堂教学，并通过**录像回放**进行自我分析和集体评议的训练方式。

## 典型流程（5步循环）

1. **讲解示范** — 导师讲解某项技能（如导入技能、板书技能），播放示范录像
2. **微型备课** — 学员针对单项技能，设计5-10分钟的微型课
3. **微型教学** — 面对少量学生（或同伴扮演）进行实际授课，全程录像
4. **反馈评价** — 回放录像，自我评价 + 同伴互评 + 导师点评
5. **重教改进** — 根据反馈修改设计，再次授课，直到达标

## 关键特点

| 特点 | 说明 |
|------|------|
| **规模小** | 人数少、时间短、内容单一，降低试错成本 |
| **技能聚焦** | 每次只训练1-2个教学技能（如提问、讲解、板书），不贪多 |
| **即时反馈** | 录像回放是最直观的"镜子"，很多自己意识不到的细节一看便知 |
| **安全环境** | 没有真实课堂的压力，可以大胆尝试、反复练习 |
| **可循环** | 教→看→评→改→再教，形成闭环 |

## 在教资考试中的应用

教资面试结构化问答中可能这样考：

> **"你怎么看待微格教学在提升教师教学能力中的作用？"**

**答题要点**：
- 微格教学让教学技能训练**可视化、可量化、可重复**
- 体现了"**实践—反思—再实践**"的教师专业成长路径
- 符合**教师教育一体化**理念，缩短从理论到实践的转化周期
