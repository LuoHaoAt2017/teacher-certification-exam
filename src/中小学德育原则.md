# 中小学德育原则

> 德育原则：教师对学生进行思想品德教育时必须遵循的基本准则。

## 8条原则一览

| 原则 | 核心含义 | 一句话 |
|------|----------|--------|
| **①导向性** | 坚持正确的政治方向 | 品德教育要有"指南针" |
| **②疏导性** | 讲道理、疏通引导，不压服 | 晓之以理，导之以行 |
| **③尊重与严格要求结合** | 尊重人格 + 严格约束 | 爱而不纵，严而有格 |
| **④教育一致性与连贯性** | 校内外、各科教师形成合力 | 五指并拢，拳头才有力 |
| **⑤因材施教** | 根据年龄、个性差异施教 | 一把钥匙开一把锁 |
| **⑥知行统一** | 理论认识 + 行为实践结合 | 说到做到，知行合一 |
| **⑦依靠积极、克服消极** | 发扬优点、克服缺点 | 用闪光点照亮阴影 |
| **⑧集体教育与个别教育结合** | 集体影响个人，个人影响集体 | 一朵花不是春天 |

---

## 四组记忆法

### 第一组：定调子
**①导向性 + ②疏导性** → 方向要对，方式要对

> 场景：班主任开班会——先讲"我们要做什么样的人"（导向），再和学生一起讨论分析（疏导），不训话、不压服。

**口诀**："导而不堵，疏而不压"

### 第二组：定规矩
**③尊重+严格 + ④一致性** → 师生关系和家校关系

> 场景：学生犯错了——先冷静说"我理解你"（尊重），再明确指出"这个行为不对"（严格），然后叫家长统一口径（一致）。

**口诀**："尊重不是纵容，一致才能有效"

### 第三组：看人头
**⑤因材施教 + ⑥知行统一** → 不同的人，不同的教法

> 场景：小李爱动手 → 让他做实验（因材），做完还要写反思（知行）；小张能说会道 → 让他上台讲道理（因材），讲完还要去实践（知行）。

**口诀**："因材看人下菜碟，知行说做两条腿"

### 第四组：讲方法
**⑦长善救失 + ⑧集体+个别** → 借力打力

> 场景：班里有个调皮鬼——老师发现他组织能力强（积极因素），让他当活动组长（扬长补短），全班配合他（集体影响），他慢慢变好了（个别进步）。

**口诀**："长善点亮闪光点，集体带个人"

## 完整速记表

| 组 | 原则 | 场景关键词 | 一句话 |
|----|------|-----------|--------|
| 定调子 | ①导向 + ②疏导 | 方向、讨论 | 定好方向好好说 |
| 定规矩 | ③尊重严格 + ④一致 | 犯错、家校 | 尊重底线不矛盾 |
| 看人头 | ⑤因材 + ⑥知行 | 个性、实践 | 看人下菜、说到做到 |
| 讲方法 | ⑦长善 + ⑧集体个别 | 扬长、借力 | 用好优点带好团队 |

### 一条顺口溜

> 导疏定方向，尊严要得当，
> 一致聚合力，因材看人强。
> 知行两条腿，长善扬所长，
> 集体带个人，八条心中藏。
