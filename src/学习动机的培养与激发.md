# 学习动机的培养与激发

## 学习动机的培养

> 从无到有，让学生"想学"

| 策略 | 要点 |
|------|------|
| **①设置合理目标** | 目标具体、可实现、有挑战性，近期目标与远期目标结合 |
| **②激发内在需要** | 将学习内容与学生的兴趣、生活经验联系起来，满足求知欲 |
| **③树立榜样** | 利用同伴榜样、教师自身的示范作用，激发模仿和上进心 |
| **④利用迁移** | 已有兴趣迁移到新领域（如将游戏兴趣迁移到编程学习） |
| **⑤正确归因** | 引导学生将成败归因于努力（内部可控因素），避免习得性无助 |

## 学习动机的激发

> 从有到强，让学生"持续学"

| 策略 | 要点 |
|------|------|
| **①创设问题情境** | 制造认知冲突，让学生产生"想知道"的迫切感 |
| **②控制动机水平** | 耶克斯-多德森定律：任务难度适中时，中等动机水平效率最高 |
| **③及时反馈** | 让学生知道学习结果，正面反馈强化正确行为 |
| **④合理奖惩** | 奖励为主、惩罚为辅，注重精神奖励，避免过度物质化 |
| **⑤竞争与合作结合** | 适当竞争激发斗志，合作学习培养互助精神，避免恶性竞争 |

## 易混淆点

- **培养** = 建立动机（从无到有的过程）
- **激发** = 调动已有动机（从弱到强的过程）

考试中两者常一起考查，注意区分各自的策略侧重点。
