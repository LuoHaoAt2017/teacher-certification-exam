# 知识、能力、品格三者区别

## 1. 知识
- 定义：对事实、概念、原理、规律的认识。
- 重点："知道什么"。
- 示例：知道计算机网络的基本结构、理解教学设计四要素。
- 考试形式：选择题、简答题中常考记忆与理解类内容。

## 2. 能力
- 定义：运用知识解决问题的实际技能与方法。
- 重点："会做什么"。
- 示例：能设计教学方案、能分析教学材料、能组织课堂讨论。
- 考试形式：教学设计题、案例分析题、材料分析题常考能力类表现。

## 3. 品格
- 定义：稳定的道德品质、价值取向和人格特征。
- 重点："成为怎么样的人"。
- 示例：责任感、公正意识、合作精神、敬业态度。
- 考试形式：德育题、职业道德题、材料题中多考品格层面。

## 4. 三者关系与辨析
- 关系：知识是基础，能力是桥梁，品格是目标。
- 区别：知识是“内容”，能力是“方式”，品格是“价值”。
- 形成路径：知识可以传授，能力需要训练，品格需要熏陶。
- 完整教育目标：同时包含知识、能力、品格，才能实现学生的全面发展。

## 5. 简答表达
- 可以说：知识决定“学什么”，能力决定“怎么做”，品格决定“成为什么样的人”。
