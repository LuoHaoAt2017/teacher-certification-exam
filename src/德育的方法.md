分类维度四：按"效力强度"分

         高强度（立竿见影，副作用大）
                 │
         品德评价法（奖惩）
                 │
         说服教育法（讲道理）
                 │
         榜样示范法（感召）
                 │
         实际锻炼法（体验）
                 │
         情感陶冶法（浸润）
                 │
         自我修养法（内省）
                 │
            低强度（见效慢，但深远持久）

  越是"外部、高强度的"方法 → 短期效果明显，但可能损害内在动机
  越是"内部、低强度的"方法 → 见效慢，但一旦形成就稳定持久
