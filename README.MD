# 二、逐周详细任务（直接照做）
## 第一阶段：教资主线 + D类渗透（第1～8周）
### 第1周：搭建框架，不深究
- 教资科目二：背**24道简答前10题**+辨析模板
- 教资科目三：背**核心素养+教学设计模板**，默写1遍
- D类：了解题型（职测D+综应D），看2套真题结构
- 每日：科二30min，科三30min，D类20min

### 第2周：科二攻坚 + 科三入门
- 科二：背完24道简答，开始刷**真题单选**
- 科三：写第1篇完整教案（Python基础）
- D类：职测——**判断推理**基础技巧
- 每日：科二40min，科三30min，D类25min

### 第3周：科二强化 + 科三实操
- 科二：刷辨析真题，整理错题
- 科三：写第2篇教案（if条件语句）
- D类：职测——**言语理解**基础
- 每日：科二40min，科三30min，D类25min

### 第4周：科一启动 + 科三巩固
- 科一：背**三观+职业道德**模板，看作文结构
- 科二：材料题万能句+真题单选二刷
- 科三：写第3篇教案（for循环/列表）
- D类：综应D——看**教育案例题**答题结构
- 每日：科一20min，科二40min，科三30min，D类20min

### 第5周：科二滚动背诵 + 科三提速
- 科二：简答循环背诵，不遗忘
- 科三：写第4篇教案（信息系统/网络）
- D类：职测——**数量分析/资料分析**入门
- 每日：科二40min，科三30min，D类30min

### 第6周：科一作文 + 科三收尾
- 科一：写1篇真题作文，练结构
- 科二：真题整套客观题（选择+辨析）
- 科三：写第5篇教案（信息安全）
- D类：综应D——背**班级管理/后进生**套话
- 每日：科一30min，科二40min，科三30min，D类20min

### 第7周：三科整合 + D类保持
- 科一：材料题模板背熟
- 科二：整套真题限时1套
- 科三：真题选择+案例分析各1套
- D类：职测每周30题，保持手感
- 每日：三科1.5h，D类40min

### 第8周：全真模拟 + 查漏补缺
- 教资：整套模拟1次（按考试时间）
- 科二：薄弱简答再背一轮
- 科三：教案模板默写2遍
- D类：不攻新内容，只维持手感
- 每日：教资2h，D类30min

## 第二阶段：教资终极冲刺（第9周 · 9.1–9.12）
**完全暂停D类新知识，只保教资稳过**
- 科二：简答狂背+真题错题复盘
- 科三：教案+案例套话滚瓜烂熟
- 科一：作文框架+材料题模板
- 每2天做1套教资真题整套
- 考前1天：只背简答+模板，不刷新题

---

## 第三阶段：全职冲D类教师编（第10～17周）
### 第10周（9.14–9.20）：D类体系搭建
- 职测D：系统学**判断+言语+策略选择**
- 综应D：辨析题+案例题答题模板
- 利用科二基础，快速衔接德育、教学、班级管理

### 第11周（9.21–9.27）：职测刷题为主
- 职测：每天1组题（30～40题）
- 综应：每天1道案例分析，规范表达
- 重点：**策略选择**（送分，必须全对）

### 第12周（9.28–10.4）：综应大题突破
- 综应：**教育方案设计**（班会/活动）模板背熟
- 职测：数量/资料针对性练简单题
- 整理综应万能句，形成自己的话术

### 第13周（10.5–10.11）：真题套卷训练
- 职测D：整套真题限时1套
- 综应D：整套真题限时1套
- 复盘时间分配，优化做题顺序

### 第14周（10.12–10.18）：弱项攻坚
- 职测：补判断/言语错题
- 综应：辨析题+案例题再练3套
- 背诵综应高频教育理论

### 第15周（10.19–10.25前一周）：全真模考
- 每2天1套职测+1套综应
- 严格按考试时间：上午职测，下午综应
- 只做真题，不做模拟题

### 第16～17周（最后10天）：复盘+背诵
- 职测：只看错题，不刷新难题
- 综应：背诵模板+套话，默写活动设计
- 保持状态，不熬夜、不突击偏题
